# KNN Code and Data

This repository contains datasets and R Markdown code completed as part of my coursework in Statistics and Machine Learning.

## Repository Contents

- **knn.Rmd** – R Markdown file containing K-Nearest Neighbors (KNN), exploratory data analysis, and regression exercises completed in R.
- **vgsales.csv** – Video game sales dataset used for data exploration, visualization, and statistical analysis.
- **BP.csv** – Blood pressure dataset used for multiple linear regression modeling and prediction.
- **README.md** – Project documentation.
